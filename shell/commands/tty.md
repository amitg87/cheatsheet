# tty

    know your terminal name
    The tty utility writes the name of the terminal attached to standard input to standard output.
    It prints ‘not a tty’ if standard input is not a terminal.

## Synopsis

`tty [OPTION]...`

## Example

* On Gnome-Terminal

> tty

    /dev/pts/8

* On Terminal-1 -> Alt-Ctrl-F1

> tty

    /dev/tty1

* On Mac

> tty

    /dev/ttys006
